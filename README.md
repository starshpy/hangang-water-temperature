# 🌊 지금 한강은 어때?

**"지금 한강은 어때?"**는 실시간 한강 물 온도를 모니터링하고 공유하는 파이썬 기반의 응용 프로그램입니다. 단순히 데이터를 수치로 보여주는 것을 넘어, 일상에 지친 사람들에게 작지만 강렬한 메시지를 전달하기 위해 만들어진 '용기를 위한 캠페인' 프로젝트입니다.

---

## 💡 제작 배경 (Motivation)

사람들은 문득 궁금해합니다. "지금 한강 물은 얼마나 차가울까?" 

단순한 호기심에서 시작된 질문일 수도 있지만, 그 이면에는 일상의 무게를 견디고 있는 누군가의 고단함이 섞여 있을지도 모릅니다. 이 프로그램은 그러한 고민을 마주하고 있는 이들에게 **"뛰어내리는걸 망설이시나요? 행동으로 실천하세요."**라는 짧고 강한 문구를 통해, 고민의 굴레를 벗어나 새로운 용기를 내길 바라는 응원의 마음을 담아 제작되었습니다.

## 🚀 주요 기능 (Features)

* **실시간 온도 확인:** 한강 물 온도 측정 API를 사용하여 현재 수온을 실시간으로 보여줍니다.
* **무설치 간편 실행:** 별도의 파이썬 환경 설정이나 설치 과정 없이, `.exe` 파일 하나로 즉시 실행 가능합니다.
* **응원 메시지 캠페인:** 프로그램을 실행할 때마다 당신의 행동을 응원하는 문구를 함께 제공합니다.

## 📥 실행 방법 (How to use)

1. 이 저장소의 **Release** 또는 파일 목록에서 `.exe` 파일을 다운로드합니다.
2. 다운로드한 `.exe` 파일을 실행합니다.
3. 실시간 한강 온도와 함께, 당신을 향한 작은 응원을 확인하세요.
