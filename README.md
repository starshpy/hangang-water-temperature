# 🌊 지금 한강은 어때?

"지금 한강은 어때?"는 실시간으로 한강 물 온도를 확인할 수 있게 도와주는 프로그램입니다.

---

## 💡 제작 배경 (Motivation)

사람들은 문득 궁금해합니다. "지금 한강 물은 얼마나 차가울까?" 

단순한 호기심에서 시작된 질문일 수도 있지만, 그 이면에는 일상의 무게를 견디고 있는 누군가의 고단함이 섞여 있을지도 모릅니다. 이 프로그램은 그러한 고민을 마주하고 있는 이들에게 **"뛰어내리는걸 망설이시나요? 행동으로 실천하세요."**라는 짧고 강한 문구를 통해, 고민 하지 말고 행동으로 실천할수 있게 새로운 용기를 내길 바라는 응원의 마음을 담아 제작되었습니다.

## 🚀 주요 기능 (Features)

* **실시간 데이터 확인:** 실시간 한강 및 주변 하천의 물 온도와 pH 농도를 제공합니다.
* **무설치 간편 실행:** 별도의 환경 설정 없이 `.exe` 파일 하나로 즉시 실행 가능합니다.

## 📥 실행 방법 (How to use)

1. 이 저장소에서 `지금 한강은 어때.exe` 파일을 다운로드합니다.
2. 다운로드한 `.exe` 파일을 실행합니다.
3. 실시간 수온 정보와 함께, 당신을 향한 작은 응원을 확인하세요.
