# 🌊 지금 한강은 어때?

**"지금 한강은 어때?"**는 실시간 한강 물 온도를 모니터링하고 공유하는 파이썬 기반의 프로그램입니다. 단순히 데이터를 수치로 보여주는 것을 넘어, 일상에 지친 사람들에게 작지만 강렬한 메시지를 전달하기 위해 만들어진 캠페인 성격의 프로젝트입니다.

---

## 💡 제작 배경 (Motivation)

사람들은 문득 궁금해합니다. "지금 한강 물은 얼마나 차가울까?" 

단순한 호기심에서 시작된 질문일 수도 있지만, 그 이면에는 일상의 무게를 견디고 있는 누군가의 고단함이 섞여 있을지도 모릅니다. 이 프로그램은 그러한 고민을 마주하고 있는 이들에게 **"뛰어내리는걸 망설이시나요? 행동으로 실천하세요."**라는 짧고 강한 문구를 통해, 고민의 굴레를 벗어나 새로운 용기를 내길 바라는 응원의 마음을 담아 제작되었습니다.

## 🚀 주요 기능 (Features)

* **실시간 온도 확인:** 한강 물 온도 측정 API를 기반으로 최신 수온 데이터를 정확하게 가져옵니다.
* **무설치 간편 실행:** 파이썬 환경 설정이 필요 없는 `.exe` 실행 파일로, 파일 하나만 실행하면 즉시 정보를 확인할 수 있습니다.
* **메시지 캠페인:** 프로그램 실행 시, 고민을 멈추고 능동적인 변화를 이끌어낼 수 있는 응원 메시지를 함께 표시합니다.

## 📥 실행 방법 (How to run)

본 프로그램은 별도의 설치 과정이 필요 없습니다.

1. 프로젝트의 페이지에서 실행 파일(`지금 한강은 어때.exe`)을 다운로드합니다.
2. 다운로드한 파일을 실행합니다.
3. 프로그램이 시작되면 실시간 한강 수온을 확인하실 수 있습니다.
